# 四元数板子（Quaternion Templates）

## 1) getroot3.sage
在有限域 GF(p) 上求四元数三次根：给定 Q，求所有 q 使得 q^3 = Q。

### 运行环境
必须使用 Sage：

```
sage getroot3.sage
```

用法
编辑 __main__ 中的参数：

p: 素数模数

Q = (w, x, y, z): 目标四元数

运行后输出所有解 q（去重）。

备注
该脚本通过范数与向量部分构造候选，再校验 q^3 == Q。

若无解，返回空列表 []。

## 一个小提醒

- `p` 必须是素数（你代码用 `GF(p)`，非素数会出问题或不符合预期）
- 结果数量可能 >1（有限域里多根很正常），所以你用 `all=True` 是对的
