## 最佳折中：允许你“事后再补知识点”，但不强迫你开局分类

你的工作流可以是：

1. 看到题 / 看WP：丢进 `00_inbox/<YYYY>/<alias>/<slug>/`
2. 真开始做：挪到 `10_working/...`
3. 跑通 exp + 写明白：挪到 `20_done/...`
4. 觉得能沉淀套路：补 tags + 抽工具到 `02_tools`，再挪到 `30_polished/...`

执行下面命令来生成对应文件夹，在cmd里面输入

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -Command "$year=Read-Host 'year'; $alias=Read-Host 'alias'; $slug=Read-Host 'slug'; $base='00_writeups\00_inbox'; $safeAlias=($alias.Trim() -replace '[^\w\-.]','-'); $safeSlug=($slug.Trim() -replace '[^\w\-.]','-'); $dir=Join-Path $base (Join-Path ($year+'_'+$safeAlias) $safeSlug); New-Item -ItemType Directory -Force -Path $dir | Out-Null; $link=Join-Path $dir 'link.txt'; if(!(Test-Path $link)){ 'source: ' | Set-Content -Encoding UTF8 $link }; $readme=Join-Path $dir 'README.md'; if(!(Test-Path $readme)){ $tpl='03_templates\writeup_template.md'; if(Test-Path $tpl){ Copy-Item $tpl $readme -Force } else { @('# '+$safeSlug,'','- **Year**: '+$year,'- **Event**: '+$alias,'- **Tags**: ','','## Problem','(题面/关键参数/附件说明)','', '## Idea','- 入口特征：','- 关键等式/结构：','- 为什么可行：','', '## Exploit','```bash','python exp.py','```','', '## Notes','- 坑点：','- 可复用板子：') | Set-Content -Encoding UTF8 $readme } }; Write-Host ('OK -> '+$dir)"

```

你完全不需要在第 1 步就想清楚“这题属于啥知识点”。

## 你问的“2023_xx比赛_题目名”是否推荐？

不推荐把比赛全称写进目录名，原因就是你说的：太乱、不可控。
 我推荐你把它改成“规范化三段式”：

✅ 推荐：
 `YYYY/CTF_ALIAS/challenge_slug`

❌ 不推荐：
 `2023_xx比赛_题目名`（因为 xx比赛会越来越多写法）

比赛全称放 README 里就够了，目录只用别名。

## 给你一个“你不会卡”的最终答案

**目录用比赛别名 + 年份（快）**
 **知识点用 tags + rg 搜索（好找）**
 **阶段用 stage 文件夹（看一眼就知道状态）**

这套能同时满足你两种需求，而且不会拖慢刷题。



补全空目录，让空目录也能上传到github

```
Get-ChildItem -Directory -Recurse -Force |
  Where-Object { $_.FullName -notmatch '\\\.git(\\|$)' } |
  Where-Object {
    # 目录里若没有任何文件/子目录，或只有 .gitkeep，则视为“需要补”
    $items = Get-ChildItem -LiteralPath $_.FullName -Force -ErrorAction SilentlyContinue
    ($items.Count -eq 0) -or (
      $items.Count -eq 1 -and $items[0].Name -eq '.gitkeep'
    )
  } |
  ForEach-Object {
    $keep = Join-Path $_.FullName '.gitkeep'
    if (-not (Test-Path -LiteralPath $keep)) {
      New-Item -ItemType File -Path $keep | Out-Null
      Write-Host "[+] add .gitkeep -> $($_.FullName)"
    }
  }
```
