如果目录按知识点命名，会逼你**在开题前就先分类** → 会严重降低刷题意愿。

所以：**目录只保证快，知识点用 tags 补上。**

在每题的 `README.md` 顶部加一个固定小头（你以后全仓库搜索就靠它）：

```
# crypto-rsa-reused-nonce

- CTF: DASCTF 2024
- Date: 2026-01-04
- Status: done
- Tags: rsa, nonce, lattice, coppersmith
- Key Trick: 复用nonce -> 构造多项式 -> 求根
```

### 然后你找“我以前写过 lattice 的是哪题”就一句命令

在仓库根目录：

```
rg "Tags:.*lattice" 00_writeups -n
```

或者你搜关键句：

```
rg "coppersmith|small root" 00_writeups -n
```

**这比记比赛名靠谱太多**。
