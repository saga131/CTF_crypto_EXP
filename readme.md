
	- ✅ 资产仓库
	- ❌ 不是比赛提交仓库
	- ❌ 不追求“每题都放”，只放**有价值的**

## 🧩 已收录方向

### 1️⃣ Gröbner Basis（多项式消元）

- 模 n 的多项式系统
- 消辅助变量（如 a）
- 解目标变量（如 `x = m² mod n`）
- 面向 **CTF 自动化消元**

📂 `grobner_basis/`

------

### 2️⃣ 四元数（Quaternion）

- 有限域 `GF(p)` 上的四元数运算
- 非交换代数
- 四元数幂、开根、结构性攻击

📂 `四元数/`

- `getroot3.py`：四元数三次根板子（需 Sage）

## ⚙️ 运行环境说明

- **强烈依赖 SageMath**

- 推荐运行方式：

	```
	sage -python script.py
	```

- 普通 `python` 解释器 **不保证可运行**
